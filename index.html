Project Overview

The Estimate App is a comprehensive financial estimation tool designed for users to compute their required savings for a time period in order to purchase a certain product or reach a goal.

Technologies Used
Java Firebase Android Studio

Key Features

  • Dynamic Calculations: Real-time calculation of totals as items are added or modified.
  • Support for Various Products: Wide selection of products to create a goal for.
  • User History: Users can create accounts to help with tracking.
  • Item Database: Reusable product/service database with history and quick-add functionality.
  • Mobile Base Design: Fully functional on mobile devices.

Outcome & Results

The Estimate App has helped users save more efficiently, with many reporting a significant reduction in time spent planning and tracking their savings goals.
